# reports

This directory stores generated logs and CSV reports.

Typical contents:
- `bot.log`
- `report.csv`
- `backtest_report.csv`

What goes here:
- runtime bot logs;
- closed trade reports;
- backtest output files.

What to do with it:
- use these files for analysis, debugging, and review;
- do not commit personal or environment-specific report files;
- safe to clean when you want a fresh run, if you no longer need the history.

This directory is intentionally kept in git only as a documented placeholder.
